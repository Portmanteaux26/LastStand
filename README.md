# Last Stand

## Installation

```
pip install -r requirements.txt
python main.py
```

## Controls

| Input              | Action |
|--------------------|---|
| WASD / Arrow Keys  | Move |
| Mouse              | Aim |
| Left Click / Space | Shoot |
| E                  | Select upgrade |
| Space              | Start game |
| Escape             | Quit |

## Goal

Survive endless waves of enemies. Each wave gets bigger and faster. Clear all enemies in a wave to earn a short break before the next one. How far can you get?
